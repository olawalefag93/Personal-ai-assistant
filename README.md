# OlawaleAI

A personal AI assistant built with FastAPI, RAG (Retrieval-Augmented Generation),
SQLite memory, tool calling, and a custom web UI — all running locally from my
external SSD.

## Features

- FastAPI backend
- RAG over local notes (ChromaDB)
- Per-session conversation memory (SQLite)
- Multiple modes: Security / Founder / Creator
- Tool calling (system info & log analysis)
- Simple web UI

## Tech Stack

- Python
- FastAPI
- OpenAI API (chat & embeddings)
- ChromaDB
- SQLite
- HTML/CSS/JS frontend
